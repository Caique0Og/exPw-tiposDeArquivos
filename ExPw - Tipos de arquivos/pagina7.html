Tipos de arquivos:

  1. TXT

    2. Descrição: Arquivo de texto simples, contendo apenas caracteres ASCII. Utilização: Ideal para armazenar notas, scripts, código-fonte simples, etc. Exemplos: lista de compras, poema, código Python básico.

  2. EXE

    3. Descrição: Arquivo executável, contendo instruções para o computador executar um programa. Utilização: Para instalar e executar programas, jogos, aplicativos, etc. Exemplos: instalador do Windows, jogo "Minecraft", aplicativo de edição de fotos.

  3. DLL

    4. Descrição: Biblioteca de vínculo dinâmico, contendo funções e recursos que podem ser utilizados por outros programas. Utilização: Para adicionar funcionalidades a programas, sem a necessidade de incluí-las no código-fonte. Exemplos: biblioteca gráfica para jogos, biblioteca de criptografia.

  4. ZIP

    5. Descrição: Arquivo compactado, contendo um ou mais arquivos em um único pacote para reduzir o tamanho total. Utilização: Para facilitar o compartilhamento de arquivos, reduzir o tempo de download e economizar espaço em disco. Exemplos: pacote de fotos para envio por e-mail, download de software.

  5. PDF

    6. Descrição: Portable Document Format, um formato universal para documentos que preserva o layout e formatação original. Utilização: Para compartilhar documentos que podem ser visualizados em qualquer dispositivo com o software Adobe Acrobat Reader ou similar. Exemplos: currículo, contrato, manual de instruções.

  6. DOC E DOCX

    7. Descrição: Formatos de documentos do Microsoft Word. O DOC é mais antigo, enquanto o DOCX é o formato padrão mais recente. Utilização: Para criar e editar documentos com formatação rica, como texto, imagens, tabelas, etc. Exemplos: carta formal, relatório de pesquisa, folheto informativo.

  7. PPT E PPS

    8. Descrição: Formatos de apresentações do Microsoft PowerPoint. O PPT é usado para editar apresentações, enquanto o PPS é usado para exibi-las em modo de slide show. Utilização: Para criar e apresentar slides com conteúdo textual, imagens, vídeos, animações, etc. Exemplos: palestra, apresentação de marketing, treinamento.

  8. SLX e XLSC

    9. Provavelmente formatos antigos do Microsoft Excel. O formato padrão atual é o XLSX. Provavelmente não são compatíveis com versões recentes do Excel e podem exigir conversores para abri-los.

  9. JPG

    10. Joint Photographic Experts Group, formato de imagem com compressão lossy (perde qualidade com a compressão) popular para fotos devido ao bom balanço entre tamanho e qualidade.

  10. GIF e PNG

    11. Graphics Interchange Format, formato de imagem que permite animações simples e suporta transparência.

    Portable Network Graphic, formato de imagem com compressão lossless (sem perda de qualidade) popular para imagens com texto, logos, ícones e gráficos com transparência.

  11. HTML

    12. HyperText Markup Language, a linguagem de marcação usada para criar páginas web. Define a estrutura e o conteúdo da página.

  12. MP3, WAV, MID

    13. MPEG-1 Audio Layer III, formato de áudio comprimido com alta qualidade de som, popular para músicas devido ao tamanho reduzido.

    Waveform Audio Format, formato de áudio digital sem compressão, comum para arquivos de áudio de alta fidelidade.

    Musical Instrument Digital Interface, formato de arquivo MIDI, que armazena instruções para instrumentos musicais eletrônicos.

  13. AVI, MPG, WMV, MOV

    14. Audio Video Interleave, formato de arquivo antigo para vídeos no Windows.

    MPEG (Moving Picture Experts Group), formato genérico para compressão de vídeo. Pode se referir a diversos formatos específicos como MPEG-1, MPEG-2.

    Windows Media Video, formato de vídeo desenvolvido pela Microsoft.

    QuickTime Movie, formato de arquivo de vídeo desenvolvido pela Apple.
