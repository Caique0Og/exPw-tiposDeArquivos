Tipos de arquivos!

  • CSS

    • Cascading Style Sheets, define a aparência das páginas web, controlando cores, fontes, layout, etc.

  • JS

    • JavaScript, linguagem de script usada para adicionar interatividade e comportamento dinâmico às páginas web.

  • SVG

    • Scalable Vector Graphics, formato baseado em vetores para imagens vetoriais, que mantêm a qualidade em qualquer tamanho.

  • PHP

    • Hypertext Preprocessor, linguagem de script usada principalmente para desenvolvimento web do lado do servidor para gerar conteúdo dinâmico.

  • ASPX

    • Active Server Pages.NET, tecnologia Microsoft para desenvolvimento web do lado do servidor similar ao PHP.

  • SQL

    • Structured Query Language, linguagem usada para gerenciar dados em bancos de dados relacionais.

  • PSA

    • Usado pelo software Seabird SBE Data Processing para armazenar configurações de programa. Esses arquivos contêm informações como o nome do arquivo de entrada, caminho de saída e parâmetros específicos do módulo utilizado (por exemplo, conversão de dados).

  • CDR AI

    • São formatos de arquivos vetoriais usados em softwares de vetorização como: CDR: CorelDRAW AI: Adobe Illustrator Ideais para criar logos, ícones, ilustrações e artes gráficas que mantêm a qualidade em qualquer tamanho.

  • XML

    • Linguagem flexível para troca de dados entre sistemas. Usa tags para definir estrutura e conteúdo dos dados (legível por humanos e programas). Exemplos: configuração de programas, armazenamento de dados para faturas eletrônicas.

  • JSP, CFM

    • Tecnologias para desenvolvimento web dinâmico do lado do servidor. Combinam HTML com código para gerar conteúdo personalizado. Diferença: JSP: usa linguagem Java. CFM: usa linguagem ColdFusion (Adobe). Exemplos: lojas virtuais, sistemas de login seguros.

  • TIF

    • Formato de imagem de alta qualidade usado profissionalmente. Suporta alta resolução, profundidade de cor e camadas (para edição). Exemplos: fotografia digital profissional, imagens para impressão gráfica. Desvantagem: arquivos maiores que JPG ou PNG.

  • FLA e SWF

    • Formatos relacionados ao Adobe Flash (tecnologia em desuso). FLA: arquivo nativo do Flash para criar e editar animações vetoriais interativas. SWF: formato exportado para incorporar animações em páginas web. Recomenda-se usar tecnologias alternativas como HTML5 para animações web modernas.

  • DWG

    • Formato nativo do AutoCAD, software de desenho assistido por computador (CAD). Usado por engenheiros, arquitetos e designers para criar projetos técnicos com alta precisão.
